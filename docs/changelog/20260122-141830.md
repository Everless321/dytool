1. **DatabaseService.swift**
   - `addUser` 函数新增 `nickname` 参数
   - 支持保存用户自定义名称到数据库

2. **UserListView.swift**
   - `AddUserSheet`: 新增"用户名称 (可选)"输入框

### 功能说明
- 添加用户时可以设置自定义名称
- 编辑用户时可以修改名称
- 名称为空时自动从 URL 提取用户 ID 显示
- 下载时显示用户名称（已有 displayName 支持）
