# 修复 ForEach 重复 ID 错误

## 任务描述
修复视频浏览页面 ForEach 中出现重复 ID 的问题

## 待办事项
- [x] 分析重复 ID 的原因
- [x] 修复 loadAllVideosForFiltering 函数
- [x] 添加去重机制
- [x] 测试验证

## 问题分析

错误信息：
```
ForEach: the ID /path/to/video.mp4 occurs multiple times within the collection
```

原因：
1. `loadAllVideosForFiltering()` 函数没有检查 `isLoadingMore` 状态
2. 多个 onChange 处理器可能同时触发，导致并发加载
3. 没有去重机制，相同视频可能被多次添加到数组

## 执行结果

### 修改文件

**VideoGalleryView.swift**

1. 添加 `loadedPathSet: Set<String>` 状态变量，追踪已加载的视频路径

2. 在 `loadVideos()` 中重置 `loadedPathSet`

3. 在 `loadMoreVideos()` 中添加去重逻辑：
   - 过滤掉已在 `loadedPathSet` 中的视频
   - 将新视频路径添加到 `loadedPathSet`

4. 在 `loadAllVideosForFiltering()` 中：
   - 添加 `guard !isLoadingMore else { return }` 防止并发调用
   - 添加相同的去重逻辑
