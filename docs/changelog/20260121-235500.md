## 功能说明

### 1. 标签显示
- 网格视图：显示分类标签和前3个分析标签
- 列表视图：显示分类标签和前5个分析标签
- 擦边等级角标：右上角火焰图标，颜色根据等级变化

### 2. 过滤器面板
- 点击过滤器按钮展开/收起
- 擦边等级范围滑块（0-10）
- 分类下拉选择
- 仅显示已分析视频开关
- 标签芯片选择（支持多选）
- 重置按钮

### 3. 数据模型更新
- LocalVideo 增加 analysis 属性
- 加载视频时自动附加分析数据
- 自动收集可用标签和分类列表

### 4. 新增组件
- `SexyLevelBadge` - 擦边等级角标
- `TagFilterChip` - 标签过滤芯片
